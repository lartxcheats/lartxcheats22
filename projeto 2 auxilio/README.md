# 🎮 AUXILIO LartxCheats

Aplicativo PWA para auxílio no Free Fire.

## 📱 Como Instalar no Celular

### Android (Chrome):
1. Abra o site no Chrome
2. Clique no botão "Instalar App" que aparece no topo
3. Ou vá em Menu (⋮) → "Adicionar à tela inicial"
4. O ícone "LartxCheats" aparecerá na sua tela inicial

### iOS (Safari):
1. Abra o site no Safari
2. Toque no botão de compartilhar (quadrado com seta)
3. Role para baixo e toque em "Adicionar à Tela de Início"
4. Toque em "Adicionar"

## 🚀 Como Hospedar Online

### Opção 1: Netlify (Recomendado)
1. Acesse [netlify.com](https://netlify.com)
2. Faça login
3. Arraste a pasta do projeto
4. Pronto! Link gerado automaticamente

### Opção 2: Vercel
1. Acesse [vercel.com](https://vercel.com)
2. Faça login
3. Clique em "Add New Project"
4. Faça upload dos arquivos

### Opção 3: GitHub Pages
1. Crie um repositório no GitHub
2. Faça upload dos arquivos
3. Vá em Settings → Pages
4. Ative o GitHub Pages
