# 📥 Como Baixar o Ícone do Discord

Você precisa de um ícone do Discord (512x512 pixels) para o bypass funcionar.

## Opção 1 - Download Direto (Mais Fácil)

1. Acesse: https://www.freepnglogos.com/pics/discord-logo
2. Clique com botão direito na imagem do Discord
3. "Salvar imagem como..."
4. Salve como: `discord-icon.png`
5. Coloque na pasta `vip/`

## Opção 2 - Site de Ícones

1. Acesse: https://icons8.com/icons/set/discord
2. Escolha o ícone do Discord
3. Baixe em 512x512 PNG
4. Salve como: `discord-icon.png`
5. Coloque na pasta `vip/`

## Opção 3 - Google Imagens

1. Google: "discord icon png 512x512"
2. Baixe uma imagem de boa qualidade
3. Salve como: `discord-icon.png`
4. Coloque na pasta `vip/`

Quando instalar no celular vai aparecer:
- **Nome:** Discord
- **Ícone:** Logo do Discord (roxo)
- **Descrição:** Discord - Chat for Communities

Perfeito para bypass! 🎭
